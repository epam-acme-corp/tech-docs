# Acme Tech — Platform Engineering Documentation

Documentation site for Acme Tech, the shared technology services subsidiary of Acme Corporation.

Built with [Astro Starlight](https://starlight.astro.build/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployed automatically to GitHub Pages via GitHub Actions on push to `main`.

🔗 **Live site**: https://epam-acme-corp.github.io/tech-docs/
